# Fundamentals of Parallel Computing - Lab 1

## Постановка задачі

1. Налаштувати середовище для розробки на мові C++.
2. Створити серверний додаток, який за допомогою сокетів приймає запити, виконує обрахунки та формує відповідь на них відповідно до варіанту.
3. Створити клієнтський додаток для виклику і тестування сервера. Для кожної відповіді має перевірятись її коректність. В разі усіх коректних відповідей програма завершується з кодом 0, в іншому випадку 1.

## Умови

- Для **парних варіантів**: для кожного нового з’єднання створюється окремий потік для обробки.
- Для **непарних варіантів**: використати механізм `select`, який при визначенні змін на сокеті передає завдання для обробки в окремий потік, через чергу.
- Дані передавати **рядками**, без спеціальних форматів.
- Коректна обробка некоректних даних: у разі помилки сервер повертає рядок `"invalid input"`.
- Для завдань із рядками можна обмежитися англійським алфавітом.
- Можна використовувати стандартні контейнери `std` для обчислень.

## Завдання для варіанту 5

- Повернути ряд із **10 випадкових натуральних чисел**, помножених на `n`.

## Інструкція по запуску

1. Запустити сервер (`server.exe`).
2. У новій консолі запустити клієнта (`client.exe`).
3. Клієнт запитує користувача ввести число `n`.
4. Сервер повертає 10 випадкових чисел × `n` або `"invalid input"`, якщо дані некоректні.
5. Клієнт можна закрити введенням `"exit"`.
